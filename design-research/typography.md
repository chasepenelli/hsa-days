# Typography Deep Dive

## Primary Recommendation

**Keep Lora + Inter, but refine the scale and usage rules.**

Lora is confirmed by Typewolf, Beautiful Web Type, and the Figma Google Fonts pairing guide as an excellent choice for warm editorial contexts. Its calligraphic roots and brushed curves carry emotional warmth that geometric serifs cannot. Inter's clarity makes it effortless for body text.

The opportunity is in execution — scale, weight selection, line height — not the typeface choice.

---

## Proposed Typographic Scale

| Role | Font | Size | Weight | Leading |
|------|------|------|--------|---------|
| Display | Lora | 56-72px | 400 (or 700) | 1.1 |
| H1 | Lora | 40-48px | 400 | 1.2 |
| H2 | Lora | 28-32px | 400 italic | 1.25 |
| H3 | Inter | 18-20px | 600 | 1.3 |
| Body | Inter | 16-17px | 400 | 1.6-1.65 |
| Caption | Inter | 13-14px | 400 | 1.5 |
| Pull Quote | Lora | 22-26px | 400 italic | 1.5 |

### Line Length

Target: **60-70 characters** for body text.

Emotional content reads better at the shorter end of this range. Dense text feels unsafe. Shorter lines encourage slower, more contemplative reading.

## Alternate Font Directions (if exploring beyond Lora)

| Pairing | Character | Best For |
|---------|-----------|----------|
| DM Serif Display + Inter | More expressive | Large display sizes, hero headlines |
| Playfair Display + Inter | More dramatic | Premium editorial feel |
| Crimson Pro + Inter | Most book-like | Long-form entry content, day pages |
| Source Serif 4 + Inter | Warmest body reading | Extended reading sessions |

---

## Usage Rules

- **Lora** for anything that carries emotional weight: headlines, quotes, the day title, journal prompts
- **Inter** for anything functional: navigation, labels, buttons, captions, body text
- **Lora italic** is the emotional highlight — use for subheadings, pull quotes, and the "voice" moments in the copy
- Never use Lora for small UI text (under 18px) — it loses its warmth at small sizes
- Body text should breathe: 1.6-1.65 line height, generous paragraph spacing
